## 1. 서론
제너릭은 TypeScript의 강력한 기능 중 하나로, 코드의 재사용성을 높여줍니다. 이번 글에서는 제너릭의 개념과 사용 방법에 대해 알아보겠습니다.

## 2. 제너릭의 기본 개념
제너릭은 함수나 클래스의 타입을 매개변수로 받아 다양한 타입을 처리할 수 있도록 하는 기능입니다. 이를 통해 코드의 유연성과 재사용성을 높일 수 있습니다.
```
function identity<T>(arg: T): T {
  return arg;
}

const output = identity<string>('Hello, TypeScript!');
```

## 3. 제너릭 인터페이스
제너릭을 인터페이스에 적용하여 다양한 타입을 지원하는 객체를 생성할 수 있습니다.
```
interface GenericIdentityFn<T> {
  (arg: T): T;
}

const myIdentity: GenericIdentityFn<number> = (arg) => arg;
```

## 4. 제너릭 클래스
제너릭을 클래스에 적용하여 다양한 타입을 처리할 수 있는 클래스를 만들 수 있습니다.
```
class GenericBox<T> {
  private contents: T;

  constructor(value: T) {
    this.contents = value;
  }

  public getContents(): T {
    return this.contents;
  }
}

const box = new GenericBox<string>('Hello');
console.log(box.getContents()); // Hello
```
