## 1. Git 명령어 모음

### 원격 저장소 리스트 확인

```
git remote -v
```

### 원격 저장소 추가

```
git remote add <원격 저장소 이름><원격 저장소 주소>
```

### 원격 저장소 브랜치 확인

```
git branch -r
```

### 원격 저장소 PUSH

```
git push <원격 저장소 이름> <원격 저장소 주소>
```

### 원격 저장소 URL 수정

```
git remote set-url origin <원격 저장소 url>
```
