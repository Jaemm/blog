## 프로젝트 개요
  - 프로젝트명: 전기 차도선 관제 플랫폼
  - 기간: 2023-12 ~ 2024-09
  - 요약: React, Node, AWS를 활용하여 전기차 배터리의 상태 및 사용량을 실시간으로 모니터링하고 관리하는 시스템 개발
  - 역할: 프론트엔드 및 백엔드 설계와 개발, 실시간 데이터 처리, AWS 인프라 배포
  
## 기술 스택
  - 프론트엔드: React.js, Next.js
  - 백엔드: Node.js, Express.js, AWS (EC2, Lambda, S3, DynamoDB), MySQL
  - 상태 관리: Recoil
  - 실시간 통신: Socket.IO-client
  - 데이터 시각화: Chart.js, D3.js
  - HTTP 통신: Axios

## 주요성과
  - 프론트엔드 개발

    React.js, Next.js를 사용해 배터리 상태 및 사용량을 실시간으로 확인할 수 있는 대시보드 구현
    Chart.js를 통해 사용량 통계 시각화 및 히스토리 기능 제공
    실시간 경고 및 알림 기능으로 사용자에게 배터리 이상 상태 전달

  - 백엔드 개발

    Node.js와 Express.js로 서버 개발, WebSocket을 통한 실시간 데이터 통신 구현
    AWS Lambda와 연동한 서버리스 아키텍처 구축으로 확장성 확보
    MySQL을 사용해 배터리 데이터 저장 및 RESTful API를 설계하여 프론트엔드와의 통신 처리

  - 실시간 데이터 처리

    AWS DynamoDB를 사용해 대량의 배터리 데이터를 실시간으로 처리
    비정기적 이벤트 처리 및 비상 상황 시 빠른 상태 업데이트

  - 배포 및 유지보수

    AWS EC2를 통해 배포 환경 구성, S3로 정적 파일 관리
    성능 모니터링과 자동화된 배포 파이프라인 구축
